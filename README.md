# PatientSpace

## Overview

Pytorch implementation of the paper "PatientSpace: An interpretable graph -based latent space for multimodal neuroimaging biomarker learning in Alzheimer’s Disease and Frontotemporal Dementia"

## Used modules

### Training modules

* Python 3.9.16
* Pytorch 2.0.1
* Nibabel 5.1.0
* MONAI 1.3.0

### Graph modules

* Python 3.10.15
* Nibabel 5.3.2
* NetworkX 3.4.2
